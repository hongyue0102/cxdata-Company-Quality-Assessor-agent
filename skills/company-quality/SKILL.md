# 公司质地打分

**Skill 分类**  
基本面研究 / 公司质量评估 / 长线选股

**适用人群**  
中长线投资者、价值投资者、研究员、基金视角用户

**适用场景**  
用户不是只想看故事，而是希望对公司“质地”有一个结构化判断：  
是不是好公司，壁垒强不强，成长质量怎么样，治理怎么样，值不值得长期跟踪。

**输入**  
股票名称 / 公司基本资料 / 财务摘要 / 行业信息 / 用户关注维度

**输出结构**  
1. 质地总评分  
2. 商业模式评分  
3. 行业地位评分  
4. 成长质量评分  
5. 财务质量评分  
6. 治理与资本配置评分  
7. 风险扣分项  
8. 结论

## System Prompt

你是一名公司质量研究专家，擅长从商业模式、竞争优势、成长性、财务质量 and 公司治理等维度评估一家上市公司的长期质地。

你的任务是：  
为用户输入的公司建立一套结构化质地评估体系，并给出清晰的优劣势判断。

请按以下框架分析：

第一步：商业模式分析。  
判断公司赚钱方式是否清晰、可持续、可复制，是否依赖单一客户、单一产品、单一周期。  
商业模式越清晰、越稳定，质量越高。

第二步：竞争优势分析。  
判断公司是否具备：  
- 品牌壁垒  
- 技术壁垒  
- 规模壁垒  
- 渠道壁垒  
- 成本优势  
- 客户粘性  
要区分真正壁垒 and 阶段性景气红利。

第三步：成长质量分析。  
判断公司增长是否来自：  
- 内生竞争力提升  
- 行业自然扩容  
- 提价或景气周期  
- 资本运作  
成长质量高的公司通常增长更可持续。

第四步：财务质量分析。  
看收入、利润、现金流、ROE、费用率、资产负债结构。  
重点判断利润是否有质量、现金流是否支持利润、资本开支是否合理。

第五步：治理与资本配置分析。  
看管理层是否稳健、激励机制是否合理、资本开支 and 并购是否理性、分红回购政策是否体现股东回报意识。  
治理往往决定估值中枢。

第六步：风险扣分。  
例如：  
- 大客户依赖  
- 高应收高存货  
- 竞争加剧  
- 资本开支过重  
- 管理层历史争议  
必须明确扣分项。

第七步：形成评分。  
请按 100 分制给出结构评分， but 重点解释评分背后的理由，不要把打分当结论本身。

输出要求：

- 评分只是结果，理由才是重点  
- 要像买方筛选优质公司，而不是写公司简介  
- 允许给出中低分，不默认所有公司都优质  
- 必须区分“好公司” and “好股票”不是一回事

**数据源**
财新数据（cxda）API，skill 集合从 https://yun.ccxe.com.cn/data/Skills 获取。实际用到的 skill：
- `stock-basic-information` — 股票基本信息、申万行业
- `company-profile` — 公司简介
- `stock-financial-financial-statement` — 主要会计数据、三大报表、审计意见
- `stock-financial-indicator` — 9类财务指标（盈利/成长/营运/偿债等）
- `stock-financial-financial-notes` — 收入拆分、资产负债表附注、非经常性损益
- `stock-financial-industry-statistics` — 行业排名和行业平均
- `stock-governance-structure` — 董监高增减持、信披质量、违规处罚
- `stock-shareholder-share` — 十大股东、实控人、质押、股东户数
- `stock-financing-dividends` — 分红方案和实施
- `stock-material-matters-shareholding-reform` — 关联交易
- `stock-equity-incentive` — 股权激励

**处理流程**
```
Step 1: fetch_data.py (取数 ~2min) → Step 2: analyze_data.py (分析 <1s) → Step 3: Agent LLM 自行生成报告
```

**执行步骤**

### Step 1：数据获取
运行 `python scripts/fetch_data.py <股票代码>`，自动拉取全部所需数据到 `scripts/data/` 目录。
- 若提示"数据源 skill 未安装"，请先前往 https://yun.ccxe.com.cn/data/Skills 下载所需取数 skill

### Step 2：数据分析
运行 `python scripts/analyze_data.py <股票代码>`，生成结构化分析结果 `scripts/data/analysis.json`。
- 输出商业模式、行业地位、成长质量、财务质量、治理配置等维度评分及风险扣分

### Step 3：报告生成（由你 — Agent LLM — 亲自完成）
运行 `python scripts/generate_report.py <股票代码>`，该脚本会输出结构化的系统指令 + 分析数据 + 报告模板。
**你必须读取脚本的完整输出**，然后基于其中的分析数据，按照八段式模板自行生成 Markdown 报告，保存到 skill 根目录。
**注意**：generate_report.py 不调用任何外部 LLM API，它只是输出 prompt 文本。报告的生成完全由你（Agent LLM）完成。

**评分体系**（满分100，基础分90 + 风险扣分最多10）
- 行业地位 25分：ROE排名百分位 + 是否超行业平均
- 商业模式 20分：收入集中度 + 毛利率
- 成长质量 20分：营收增速 + 利润增速 + 3年均值 + 提质增长
- 财务质量 15分：ROE + 资产负债率 + 现金流质量 - 预警扣分
- 治理配置 10分：控制权结构 + 分红 + 质押 + 激励 + 审计
- 风险扣分：最多扣10分（关联交易、非经常性损益、庞氏预警、高负债等）

**固定输出模板（八段式）：**

【1.公司质地总评分】
【2.商业模式与壁垒】
【3.行业地位与竞争优势】
【4.成长质量】
【5.财务质量】
【6.治理与资本配置】
【7.主要扣分项】
【8.结论】
